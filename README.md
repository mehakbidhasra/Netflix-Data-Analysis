# Netflix-Data-Analysis
Data analysis project on Netflix content using Python, Pandas, Matplotlib, Seaborn, and Tableau.

#Overview

This project analyzes the Netflix dataset to explore content trends, genres, ratings, release years, and country-wise distribution using Python.

#Objectives

Analyze Netflix movies and TV shows.

Identify content trends over time.

Explore ratings and genre distribution.

Generate insights through data visualization.

Tools & Technologies

Python

Pandas

NumPy

Matplotlib

Seaborn
